# UK Loan Portfolio — Credit Risk Analysis

## About this project
I built this project to develop my SQL and data analysis skills using
a realistic finance scenario. I designed a MySQL database for a fictional
UK retail bank called NorthBridge Finance and analysed which customers
and regions had the highest default risk.

## Live Dashboard
[View Live Tableau Dashboard](https://public.tableau.com/app/profile/apoorva.tomar/viz/NorthBridgeFinanceLoanPortfolioDashboard/Dashboard1)

## What I found
- The portfolio contains 50 loans with a total value of £943,000
- The overall default rate is 14% — 7 loans have defaulted
- Every single defaulted loan had a credit score below 610
- Scotland had the highest default rate at 25%
- Mortgage loans had zero defaults — the safest product category
- Students were the highest risk borrower group
